# Personal Portfolio — Dmytro Hensirovskyi

This is my personal portfolio website built with **Next.js (App Router)**, **TypeScript**, and **Tailwind/Mantine**.  
It showcases my projects in **Web Development** and **AI Automation**, as well as information about my background and services.  

🌍 Live demo: [hensirovskyi.dev](https://www.hensirovskyi.dev)  

---

## 🚀 Tech Stack
- **Frontend:** Next.js (App Router), TypeScript, React  
- **UI/UX:** Tailwind CSS, Mantine UI, Framer Motion, GSAP  
- **Deployment:** Vercel  

---

## 📂 Features
- Responsive & modern UI  
- Multi-language support (EN/DE)  
- Smooth animations and transitions  
- Dynamic project pages (works/[id])  
- Integrated contact form with Formspree (AI-powered validation planned)  

---

## 📸 Sections
- **Hero:** Introduction and CTA  
- **Latest Works:** Case studies (Websites & Apps)  
- **What I Do:** Services & skills  
- **FAQ:** Automated Q&A  
- **Pricing:** Productized offers  
- **Contact:** Interactive form + details
